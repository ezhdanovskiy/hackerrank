У Анимеша есть N пустых урн, пронумерованных от 1 до N, каждая из которых имеет бесконечную вместимость. Он выполняет M операций. Каждая операция характеризуется числами a, b, k, где a и b - это индексы урн, а k - это количество конфет, которое надо положить в каждую урну с индексом между a и b (обе границы включительно).
После того как он закончил выполнение M операций, Анимеш должен сказать, чему равно среднее количество конфет среди N урн.

**Формат входных данных**  
Первая строка содержит два целых числа N и M, разделенных пробелом.  
Каждая из следующих M строк содержит три целых числа a, b и k. Числа разделены одиночными пробелами.  

**На заметку**  
Индексы урн - это целые числа от 1 до N.

**Формат выходных данных**  
Выведите *целое значение среднего количества конфет в урнах* в единственной строке. Другими словами выведите GIF(среднее значение), где функция GIF(x) обозначает наибольшее целое меньше либо равное значения x.

**Ограничения**  
3 <= N <= 10<sup>7</sup>  
1 <= M <= 10<sup>5</sup>  
1 <= a <= b <= N  
0 <= k <= 10<sup>6</sup>  

**Пример входных данных**  

    5 3
	1 2 100
	2 5 100
	3 4 100

    
**Пример выходных данных**  

    160

**Объяснение**  
Изначально каждая урна содержит 0 конфет:  

    0 0 0 0 0  

После первой операции:  

    100 100 0 0 0  

После второй операции:  

    100 200 100 100 100  

После третьей операции:  

    100 200 200 200 100  

Всего = 800, Среднее = 800/5 = 160
